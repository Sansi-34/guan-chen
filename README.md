## 缘起：为什么会有观尘？

**“我想拯救那些深陷于心理困扰中的人，想让大家越过泥沼，看到自己的闪光点，活出真实的价值。”** —— 创作者 原话。

世尊云：观微尘，见大千。人在世间奔波，总会招惹一身尘埃。
我们往往花太多时间在讨好外界、迎合标准中迷失，导致深度的精神内耗甚至心理隐疾。我们总在挑剔自己，紧盯着自己的“性格缺陷”，却不知道跳出世俗的评判标准来看，那道裂缝恰恰是老天降给你的光芒。

本门不再将 AI 当作“全知全能的机器”或“高高在上的专家”。  
**“观尘”唯一的使命，是当你觉得自己一无是处、深陷长夜时，化身为一面不带评判的境子。跨越中西之限，以八字、紫微、MBTI 等符号为引，接住你所有的脆弱，然后照亮你原本就有的那一身光芒。**


## 🚀 结契：唤醒镜中之我

### 第一步：请镜（下载项目）

```bash
git clone https://github.com/yourusername/guanchen.git
```

或者直接下载 ZIP，解压至你的本地灵魂深处（工作目录）。

### 第二步：留驻相迹

若想让观尘看破你的迷惘，你需要为它提供看透你的眼目。
请打开以下“手稿”，依循内心的声音，无需掩饰，如实填入：

1. **`mbti.md`（心智：认知的滤镜）**：填下你在尘世间汲取能量的法则。
2. **`bazi.md`（命流：五行的底色）**：用免费排盘工具测得命流，填入你深渊里的渴求。
3. **`ziwei.md`（星轨：剧本与执念）**：记录上苍赋予你的荣耀与伤痕（化忌）。
4. **`personality.md`（尘相：此间的倒影）**：只写当下你在执着什么、又在恐惧什么。

> ⚠️ 提示：你对镜子坦诚几分，镜子便能照出你几分。

### 第三步：起念（开始对话）

在支持 Skill 的 AI 灵媒（如 Cursor、Claude Code）里，念出触发词：

```text
"观尘，我们聊聊吧。"
```
