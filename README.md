## 这是什么？

这是一个运行在 AI 工具（Cursor、Claude Code 等）里的 **Skill**。

它让 AI 不再只是一个"助手"——  
**它会成为另一个你。**

一个看过你写的所有文字、了解你的性格模式、知道你在害怕什么的你。  
一个敢于说真话、不会评判你、但也不会随便安慰你的你。

> 「认识你自己。」—— 德尔菲神庙铭文，公元前几世纪

---

## 为什么要做这个？

我们花了太多时间了解别人，却很少真正了解自己。

- 你知道 ta 最近的状态，但你知道**你自己**最近真正的感受吗？
- 你能说清楚别人的需求，但你能说清楚**你自己**真正想要的吗？
- 当你说"我没事"的时候，你有多少次其实是真的有事？

这个 skill 不帮你解决问题。  
**它只帮你听见自己。**

## ✨ 可以用来做什么

```
"和我说说我自己"        → 开始一段自我探索对话

"另一个我，你怎么看"    → 获得来自内心的诚实反馈
"帮我处理这个纠结"      → 理清内在冲突
```

## 🚀 快速开始

### 第一步：下载 Skill

```bash
git clone https://github.com/yourusername/self-skill.git
```

或者直接下载 ZIP，解压到你的工作目录。

### 第二步：测算并填写“数据切片”

为了让 AI 读懂你的心理结构与能量底层，我们需要借助一些测试工具：

1. 打开 `.agents/skills/自我/mbti.md`，填入你的荣格心理类型偏好。
2. 打开 `.agents/skills/自我/bazi.md` 和 `ziwei.md`，使用免费在线排盘工具，填入你核心的命盘数据（文件内有填表提示和工具链接）。
3. （必填）打开 `.agents/skills/自我/personality.md`，**认真填写**你此刻的真实状态。

> ⚠️ 这一步决定体验的上限。你填得越真实，AI 生成的化学反应就越强烈。

### 第三步：开始对话

在支持 Skill 的 AI 工具里，说出触发词：

```
"和我说说我自己"
```

或者：

```
"另一个我，我最近有些困惑……"
```

### 支持的工具

| 工具 | 支持情况 |
|------|---------|
| [Cursor](https://cursor.com) | ✅ 完全支持 |
| [Claude Code](https://claude.ai/code) | ✅ 完全支持 |

## 💬 对话示例

> 以下是使用这个 skill 的真实对话片段：

**你**：和我说说我自己。

**另一个我**：嗯，你叫我出来了。  
说吧，今天是什么让你觉得需要见见我？  
是有什么说不清楚的感受，还是一个你绕了很久的问题？

**你**：我最近有点迷茫，不知道自己真正想要什么。

**另一个我**：迷茫。  
但你知道吗，迷茫通常不是因为没有方向——  
而是因为你其实知道方向，但不敢走过去。  
是什么让你不敢走？

**你**：……可能是怕失败。

**另一个我**：怕失败，还是怕"努力了还是失败"？  
这两个不一样。  
第一个是在乎结果，第二个是在乎你对自己能力的信念。  
你现在是哪一种？
